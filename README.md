Wanderlust – Airbnb-Inspired Accommodation Booking Platform

Overview-
Wanderlust is a full-stack web application inspired by Airbnb that allows users to explore, create, manage, and review accommodation listings. The platform provides a seamless experience for travelers looking for places to stay while enabling property owners to showcase and manage their listings.
Built using Node.js, Express.js, MongoDB, EJS, HTML, CSS, and Bootstrap, the project demonstrates full-stack development concepts, including authentication, database management, server-side rendering, and RESTful API design.

Features:
Create, edit, and delete property listings
View detailed property information
Add and manage reviews and ratings
Secure access to protected routes
Responsive and mobile-friendly design
Server-side rendering using EJS templates
Form validation and error handling
MongoDB database integration for persistent data storage
